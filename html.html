HTML (Lesson 1)

HTML or HyperText Markup Languange is the standard markup language used to create Web pages.

HTML is written in the form of HTML elements consisting of tags enclosed in angle brackets (like <html>). HTML tags most commonly come in pairs like <h1>, and </h1>, although some tags represent empty elements and so are unpaired, for example <img>. The first tag in pair is the opening tag and the second the closing tags.

Linking in HTML (Lesson 4)

To create a link use the following tag:

<a href="link, or folder, goes here">Place the name of the link here</a>

To create a link that opens in a blank page use the following tag:

<a href="link, or folder, goes here" target="_blank">Place the name of the link here</a>"

To create a link with an image use the following tag:

<a href="link, or folder, here><img src="picture here" alt="Name of picture"><'/a>
